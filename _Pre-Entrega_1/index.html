Proyecto Final

Pre-Entrega 1

  1. El programa solicitará información al usuario.
  2. Se validarán los datos y se le construirá un mensaje de bienvenida, personalizado.
  3. Se le presentará al usuario, una lista de Frutas y Verduras. Visualizandolas por consola en una tabla.
  4. A partir de dicha lista, se le solicitará que agrupe las frutas por un lado, y las verduras por otro.
  5. A través de una función, los grupos generados por el usuario serán devueltos como dos nuevos listados. Uno de frutas, ordenados alfabéticamente de manera ascendente y otro, de verduras, ordenado alfabéticamente pero de forma descendente.
